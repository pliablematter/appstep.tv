Coding Exercises : Comparision Operators : ==

  • == (equals equals) asks the question "is the value on the left equal to the value on the right?"
  • Examples:
  • True or False? : 1 == 1 True
  • True or False? : 1 == 2 False
  • True or False? : 0 == 0 True
  • There are more comparision operators... != > < >= <= etc.
  • But we don't need to learn them right now

Coding Exercises : If

  • An If Statement lets you choose to run code only when something is true

if(1==1)
{
    NSLog(@"Hello");
}

Coding Exercises : Comments

  • Comments are hints about the code for yourself and others
  • To add a comment, just start a line with //

// This should print "Hello"
if(1==1)
{
    NSLog(@"Hello");
}

// This should NOT print "Hello"
if(1==0)
{
    NSLog(@"Hello");
}

Coding Exercises : If / Else

If / Else lets you choose to do one thing or another depending on whether something is true or false


// This should print "Hello"
if(1==1)
{
    NSLog(@"Hello");
}
else
{
    NSLog(@"Goodbye");
}

Coding Exercises : Variables

Variables let you...

  • Store pieces of information for later use
  • Give names to pieces of information

int temperature = 72;
NSLog(@"The current temperature is %d degrees", temperature);

Coding Exercises : More Variables

  • For complex types, you need to add a asterisk (*) before the variable name

// Variables
// Simple type
int temperature = 72;
// Complex type
NSString *units = @"fahrenheit";
NSLog(@"The current temperature is %d degrees %@", temperature, units);

Coding Exercises : Functions

Functions let you...

  • Give names to a series of commands
  • Reuse a series of commands multiple times

- (void) logTemperature:(int)temperature inUnits:(NSString*)units
{
    NSLog(@"The current temperature is %d degrees %@", temperature, units);
}
